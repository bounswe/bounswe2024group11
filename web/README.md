# Zenith Web App

## Install & Run

Install Node.js 18.18.0

`pnpm install` to install dependencies
`pnpm dev` to run the development server

## Build

`pnpm build` to build the project

## Technologies Used

- [Vite](https://vitejs.dev/)
- [React](https://reactjs.org/)
- [Mantine](https://mantine.dev/)
